# 图书管理系统 java887

## 项目概述

本项目是基于 Spring Boot 和 Vue 的图书管理系统，实现前后端分离的架构模式。系统以 B/S 结构为基础，采用 MVC 设计思想，提供一套完整的图书信息管理解决方案。

## 技术栈

### 系统环境
- Windows/Mac

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Mysql 8.0

### 技术框架
- Java
- Mysql
- Spring Boot
- Mybatis
- Vue

## 功能模块

系统角色分为普通用户和管理员，主要功能包括：

- 图书信息管理
- 图书类别管理
- 借阅信息管理
- 用户管理
- 修改密码
- 用户借书
- 用户还书

## 系统角色

- 普通用户：可以进行图书借阅、还书、查看个人信息等操作。
- 管理员：负责图书信息管理、用户管理等后台管理功能。

## 运行环境

- 服务器：任何支持 Java 和 Mysql 的服务器环境
- 客户端：现代浏览器

## 部署步骤

1. 准备 Java 和 Mysql 的运行环境。
2. 导入项目源码到开发工具 IDEA。
3. 使用 Maven 进行项目构建。
4. 部署前端 Vue 代码到 Web 服务器。
5. 部署后端 Spring Boot 服务。

```
项目根目录

    ├── src

```

## 核心亮点

- 采用主流的 Spring Boot 和 Vue 技术栈，保障系统性能和开发效率。
- 严谨的权限控制，确保系统安全可靠。
- 灵活的功能模块设计，满足不同角色的使用需求。
